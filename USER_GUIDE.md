# Пользовательская инструкция

## Назначение

Сервис предназначен для обезличивания документов перед их передачей во внешний контур.

## Что можно загрузить

Поддерживаются:

- `PDF`
- `DOCX`
- `TXT`
- `MD`
- `PNG`
- `JPG`
- `JPEG`
- `ZIP`

## Основной сценарий работы

### 1. Открыть сервис

После запуска откройте веб-интерфейс в браузере.

### 2. Выбрать параметры поиска

В левой панели можно выбрать категории данных, которые нужно искать:

- ФИО
- организации
- локации
- email
- телефоны
- суммы
- паспорта
- счета
- ИНН
- свой словарь

При необходимости можно включить OCR для сканов и изображений.

### 3. Загрузить документы

Добавьте один или несколько файлов через блок загрузки.

### 4. Запустить анализ

Нажмите кнопку `1. Анализировать документы`.

Сервис:

- проанализирует документы
- покажет найденные чувствительные фрагменты
- сформирует предварительный вердикт

### 5. Проверить результат

После анализа доступны:

- подсветка найденных сущностей в тексте
- таблица совпадений
- итоговый вердикт по документу

Пользователь может:

- снять галочки с фрагментов, которые не нужно вычеркивать
- вручную добавить фразы, которые система не нашла автоматически

### 6. Сформировать безопасную версию

Нажмите кнопку `2. Подтвердить и вычеркнуть`.

Сервис сформирует ZIP-архив с результатами.

## Что входит в результат

В зависимости от выбранных параметров архив может включать:

- очищенный файл в исходном формате
- `Markdown`-версию
- `Word (.docx)`-версию
- `report.json`
- `report.md`

## Дополнительные параметры

### Экспертный режим

Позволяет:

- изменить `Backend URL`
- выбрать режим движка:
  - автоматически
  - Natasha
  - Qwen

Если Qwen не настроен, система продолжит работу в fallback-режиме.

## Ограничения

- OCR на изображениях и сканах может работать менее точно, чем на PDF и DOCX с текстовым слоем
- поэтому сервис специально оставляет пользователю этап финального подтверждения перед вычеркиванием

## Рекомендации по использованию

- для демонстрации лучше использовать документы с хорошим качеством текста
- при работе со сканами проверяйте таблицу совпадений перед выгрузкой
- при необходимости используйте ручное добавление фраз
